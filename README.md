# BizForce CRM — BDA Module

A full-stack CRM application built for Business Development Associates (BDA) to manage leads, pipeline, and team performance.

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (MongoDB Atlas)
- **Authentication**: JWT
- **Drag & Drop**: @hello-pangea/dnd
- **Charts**: Recharts

## Features
- JWT Authentication
- AI-powered Lead Scoring
- Kanban Pipeline with Drag & Drop
- Convert Lead to Deal
- Sales Funnel Analytics
- Team Leaderboard
- Hot/Warm/Cold Lead Tags
